# Planned Features

# Shortcuts

  • Provide your custom shortcuts for opening/hiding the tree and other functionalities.

# Auto Real-time Update of the Tree

  • This feature is not difficult to implement, but considering the situation at OpenAI, excessive requests might lead to account suspension. The next version might include this feature, but it will require your consent and awareness of the associated risks.

# Offline Storage

  • Offline storage and management of chat contents, no longer limited by browser constraints. Hopefully, the developer can implement this soon.
