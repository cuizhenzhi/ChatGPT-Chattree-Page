# Environment Configuration

# TamperMonkey

Scripts are not recommended to be used in conjunction with other plugins (i.e., ViolentMonkey, Orange, etc.), mainly because the author uses TamperMonkey more frequently and has not adapted for other plugin platforms. If you already have the TamperMonkey extension, you can skip this step.

  • Download

Download Tampermonkey (opens new window), password: 284i

284i

  • Installation

    • Open the browser's extension management page
      • For example, in Chrome, type in the address bar and press enter:
      • chrome://extensions/
    • Enable Developer Mode (might be in the top right corner of the page, you may need to search patiently or check on Google)
    • Drag the downloaded .crx file into the page to complete installation

# Download the Script (ChatGPT Chattree)

If you have successfully installed the script, you can skip this step.

Open the script page at https://greasyfork.org/zh-CN/scripts/476683-chatgpt-chattree, and complete the installation (opens new window)

Note that the installation page will open automatically, and you need to manually click the "Install" button to confirm installation

# Results

  • First, pin Tampermonkey to the extension bar

If the extension is already pinned to the toolbar, and after opening chatgpt.com (opens new window) you see the ChatGPT Chattree activated, then the installation is successful!

If everything is fine up to this step, you can go check out the Main Features Introduction now!
