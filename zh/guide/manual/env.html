# 环境配置

# TamperMonkey(篡改猴)

脚本不建议配合其余(i.e. ViolentMonkey, Orange等)插件使用, 主要是作者使用TamperMonkey比较频繁, 没有针对其余插件平台做适配. 如果已经有TamperMonkey插件, 可以跳过

  • 下载

下载 Tampermonkey (opens new window)，密码：284i

284i

  • 安装

    • 打开浏览器的扩展管理界面
      • 例如谷歌浏览器, 在地址栏输入并回车:
      • chrome://extensions/
    • 打开开发者模式(可能在页面右上角, 需要耐心找一下或者百度)
    • 将下载好的.crx文件拖入页面中, 完成安装

# 下载脚本(ChatGPT Chattree)

如果已经安装脚本成功, 可以跳过

打开脚本网页https://greasyfork.org/zh-CN/scripts/476683-chatgpt-chattree, 完成安装 (opens new window)

注意此时会自动开启安装页面, 需要手动点击"安装"按钮确认安装

# 结果

  • 需要先将tampermonkey固定到扩展栏

如果扩展已经固定在扩展栏, 且打开chatgpt.com (opens new window)以后看到ChatGPT Chattree开启状态, 则安装成功!

如果到这一步没有问题, 就可以去看主要功能介绍啦!
